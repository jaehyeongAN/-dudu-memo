# DuDu Memo

DuDu Memo는 할 일과 메모를 관리할 수 있는 웹 애플리케이션입니다.

## 기능

- 사용자 인증 (로그인/회원가입)
- 할 일 관리 (추가, 수정, 삭제, 완료 표시)
- 하위 할 일 관리
- 메모 관리 (추가, 수정, 삭제)
- 캘린더 뷰

## 기술 스택

- Frontend: React, TypeScript, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
- 기타: Docker

## 실행 방법

'''
docker-compose build && docker-compose up
'''
